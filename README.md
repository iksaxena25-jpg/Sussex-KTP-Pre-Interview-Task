# Sussex-KTP-Pre-Interview-Task

## Accelerated Stability Study Analysis

Analysis of an accelerated stability dataset for two prototype formulations(Lot A014 and Lot B025), addressing the six questions in the exercise brief.

### Requirements
- Python 3.10 or later
- See requirements.txt

### Setup
1. Create and activate a virtual environment (optional but recommended).
2. Install dependencies:
   pip install -r requirements.txt
3. Place the supplied data file KTP_task_datasets.xls in the repository root.
   The data file is not included in this repository by design.

### Running
   jupyter lab        (or jupyter notebook)
Open Sussex_uni_final.ipynb and run all cells in order.
